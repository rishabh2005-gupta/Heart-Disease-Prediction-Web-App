❤️ HeartShield: AI-Powered Heart Disease Prediction App
A cutting-edge web application that uses machine learning to predict heart disease risk — built for impact, clarity, and user trust

📂 Project Structure
heartshield/
├── model.joblib               # Trained & saved model
├── app.py                     # Web app (Flask or Streamlit)
├── train.py                   # Model training script
├── dataset.csv                # Cleaned UCI dataset
├── requirements.txt
└── README.md                  # You are here ✨

🧪 How to Run
# 1. Clone the repo
git clone https://github.com/yourname/heartshield.git
cd heartshield

# 2. Install dependencies
pip install -r requirements.txt

# 3. Launch the app
streamlit run app.py
# or
python app.py  # for Flask

📊 Input Features
Users enter:

Age, Sex, Chest Pain Type
Resting BP, Cholesterol, Fasting Sugar
Max Heart Rate, Exercise Angina
ST Depression, ECG Results, Vessels, Thalassemia
Model outputs: High Risk 🟥 or Low Risk 🟩
